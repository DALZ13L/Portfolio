Introduction

No serious retro-gaming collection is complete without the inclusion of the iconic Commodore 64, and when it comes to emulation of this classic hardware, nothing matches the VICE64 in terms of authenticity, reliability and user-friendliness.

Nonetheless attempting to emulate 43 year old hardware on a modern PC setup can present some unique challenges, particularly when trying to boot via a custom UI such as Attract Mode. This guide will take you through the steps of how to do just that.

Before You Begin

Before you can begin you need to make sure to have the following:

  • Attract Mode
  • VICE64
  • C64 Roms in the following formats: .d64 .prg .t64 .tap .crt .p00 .g64

It is a good idea to save all these in an dedicated and organised folder for ease of future access.

Basic Set Up

In order to access and boot your ROMS within Attract Mode you need to build a landing page to house the collection, set up associations between your C64 roms and VICE64 and then point Attract Mode to the exact pathways where it can find the ROMS and the emulator.

To do this, open Attract Mode and go to Emulators > Add Emulator. Name your new emulator C64.

In the Executable field, enter the full path to the VICE emulator, for example: C:\Emulators\VICE\x64sc.exe. In the Rom Path field, enter the folder where your C64 games are stored, such as C:\Roms\C64. Set the Rom Extensions to something like d64;prg;t64 depending on your file types. Finally, in the Command Line field, enter [emulator] "[romfilename]" so Attract Mode knows how to launch each game. Once you’ve saved the settings, use the Generate Collection/Rom List option to populate your game list and complete the setup.

Note: The appearance, behavior and theme of your new C64 landing page can be fully customized with logo art, animations, preview videos, music etc, but that falls beyond the scope of this guide.

Booting Into Fullscreen

By default, VICE64 boots into windowed mode, with a hotkey toggle to trigger full screen. This is obviously not ideal when booting through a UI, where you are aiming for a seamless gaming experience.

Whilst there is no native support for default fullscreen boot, there is a way around this in Attract Mode by appending the -fullscreen flag directly at the end of the emulator path. Using our earlier example this would be C:\Emulators\VICE\x64sc.exe -fullscreen.

This ensures VICE always boots directly into fullscreen when a game is launched through the front end.

Inputting play and load commands

VICE64 is designed with high accuracy in mind, so by default it behaves like a real Commodore 64 — meaning you often have to manually type LOAD and other commands just to start a game. While that's great for authenticity, it's obviously a pain when launching games from a frontend like Attract Mode.

To make things smoother, you can configure VICE to automatically boot straight into the game

To do this, you’ll need to run x64sc.exe manually outside of Attract Mode. Once VICE is open:

  • Go to Settings > Cartridge/IO settings (or just load a cartridge via File > Attach cartridge image).
  • Then go to Settings > Settings Management.
  • Enable Autostart and disable Enable True Drive Emulation
  • Finally, go to Settings > Save Settings so VICE remembers this behavior for future launches.

This ensures that whenever VICE is launched (even via Attract Mode), it will autoload the cartridge image and start the game without asking you to type any commands.

Circumventing Load times

Another of the quirks of VICE’s commitment to authenticity is the painfully real load times — often clocking in at ten minutes or more, just like on the original hardware.

This is great for purists looking to preserve the childhood anticipation of spending ten minutes or more looking at some tantilising pixel art and listening to banging chiptune as you eagerly await the Press Start screen, but not when you just want to jump in quickly from a frontend like Attract Mode.

Luckily, VICE includes a Warp Mode feature that speeds up loading by running the emulator at 20x normal speed. However this needs to be activated and deactivated manually during gameplay by pressing Alt + W on the keyboard, and these bindings cannot be changed.

If you want to get around this without resorting to keyboard presses you’ll need to use a tool like AntiMicro or JoyToKey to bind Alt and W button to a button on your gamepad as an indirect solution.

That way, when you start a game, you can hit the mapped button to blast through loading screens and get to the action faster, all while keeping that vintage C64 feel. Just be sure to remember to toggle off again before beginning gameplay!
